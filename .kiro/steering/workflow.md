# Workflow Rules

## Git & Deployment

- **Never run `git push` or `git commit` unless explicitly asked.**
- **Never run `./scripts/deploy.sh` or any deployment command unless explicitly asked.**
- Make code changes, run tests, fix issues — but stop there.
- When work is ready, tell the user what changed and wait for them to ask to commit/push/deploy.
